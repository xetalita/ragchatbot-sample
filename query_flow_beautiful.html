Processing Flow Timeline

1

User Input Capture

JavaScript captures query, disables input, shows loading animation

  • ✓ User types query in chat input field
  • ✓ Press Enter or click Send button triggers sendMessage()
  • ✓ Input field value extracted and trimmed
  • ✓ Input/button disabled to prevent duplicates
  • ✓ User message added to chat display
  • ✓ Loading animation appears
2

API Processing

FastAPI validates request, manages session, routes to RAG system

  • ✓ POST request sent to /api/query endpoint
  • ✓ JSON payload contains query and session_id
  • ✓ FastAPI validates request with Pydantic model
  • ✓ Check if session_id exists or create new
  • ✓ Pass request to rag_system.query()
  • ✓ Handle errors with HTTPException if needed
3

AI Analysis

Claude determines if course search is needed based on query

  • ✓ Format query into AI prompt with instructions
  • ✓ Retrieve conversation history from session
  • ✓ Include available tools (search_course_content)
  • ✓ Send to Claude API with temperature=0
  • ✓ Claude analyzes query intent
  • ✓ Decides: use search tool or respond directly
4

Vector Search

ChromaDB performs semantic search on course embeddings

  • ✓ Tool manager executes CourseSearchTool
  • ✓ Optional course name resolution if provided
  • ✓ Query converted to embeddings via SentenceTransformer
  • ✓ ChromaDB searches course_content collection
  • ✓ Apply filters (course name, lesson number)
  • ✓ Return top K results with metadata & distances
5

Response Generation

Claude synthesizes final answer with context and sources

  • ✓ Tool results passed back to Claude
  • ✓ Claude processes search results as context
  • ✓ Generate comprehensive answer (max 800 tokens)
  • ✓ Session history updated with Q&A pair
  • ✓ Extract sources from tool manager
  • ✓ Format response with answer and sources
6

Display Result

Frontend renders formatted response with collapsible sources

  • ✓ Frontend receives JSON response
  • ✓ Remove loading animation
  • ✓ Parse markdown in answer using marked.js
  • ✓ Add message to chat with assistant styling
  • ✓ Create collapsible sources section if present
  • ✓ Re-enable input field and focus
